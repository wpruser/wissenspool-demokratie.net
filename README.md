# Wissenspool Demokratie

Dieses Repository enthält eine einfache GitHub-Pages-Seite für eine einseitige Linksammlung.

## Inhalt bearbeiten

Der sichtbare Inhalt der Seite liegt in `index.md`. Links und Kategorien können dort direkt als Markdown bearbeitet werden.

## Lokale Struktur

- `index.md`: Inhalt der Linksammlung
- `_layouts/default.html`: HTML-Grundlayout
- `assets/css/style.css`: einfache Gestaltung
- `_config.yml`: Jekyll/GitHub-Pages-Konfiguration

## Veröffentlichung

GitHub Pages kann dieses Repository direkt mit Jekyll bauen. Dafür in GitHub unter `Settings` -> `Pages` den Branch auswählen, aus dem veröffentlicht werden soll.
